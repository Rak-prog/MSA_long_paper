# MSA_long_paper
Code used for the MSA paper:

dicom2bids.py and do_stats.py convert dicom to bids format, process dicom information and create information for descriptive statistics

bash script are to run freesurer, quality checks, visualize results and run post fs substructure segmentation 

final_paper_revision.m and final_apper_interactions.m script to perform the lme modeling with accessory functions (the others .m files) in MatLab (diagnostic performed here)

HC_script.m and do_between_group_anopva.m for the analysis comparisons with the control group

stats_anova.R and stats_lme.R are the LME and LM modelling with effect sizes

figure_paper_revision.ipynb generate final figures for the papers (many more figures are shown to ivnestigate the results properly)
